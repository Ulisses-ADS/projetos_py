# projetos_py

Meu objetivo é que utilizando da sintaxe python desenvolver um código que consiga fazer operações matemáticas básicas 
como uma simples calculadora.
Esse código contém condicional com os if e elif, entradas de dados com input("  "), e strings para fazer um completo e bom primeiro projeto como aluno.
